# ATM Cash Wihtdrawal System - Task 1
## Description
* This project models the logical flow of an ATM withdrawal process. I've focused on creating a secure loop for PIN verification and ensuring that the withdrawal logic handles Turkish Lira (TL) denominations correctly.

# Personal Notes
* The most difficult part was nesting the "Multiple of 20" check with the "Daily Limit" check to make sure the user does not get stuck in the infinite loop if they enter a wrong amount.
